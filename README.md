# Aortic Valve Object Detection - AI CUP 2025

## 專案結構
- datasets/ : Dataset 定義
- models/ : 模型建立
- train.py : 訓練主程式
- predict.py : 推論與 submission
- utils.py : 共用工具
- requirements.txt : 相依套件

## 使用方式
1. 安裝套件: pip install -r requirements.txt
2. 訓練: python train.py
3. 推論: python predict.py
4. 提交: submission.json
